# HebrewLens - Hebrew Translation & Learning Platform

[![Donate Crypto](https://img.shields.io/badge/Donate-Crypto-f7931a?logo=bitcoin&logoColor=white)](https://commerce.coinbase.com/checkout/122a2979-e559-44b9-bb9d-2ff0c6a3025b)


## Overview

HebrewLens is a modern web application that transforms how you understand Hebrew by revealing the ancient pictographic meaning behind each letter. Unlike traditional translators that just convert words, HebrewLens provides deep insights into Hebrew's rich history and meaning.

## Features

- **Letter-by-Letter Analysis**: See each Hebrew letter's ancient pictograph and meaning
- **Interactive Alphabet**: Explore all 22 Hebrew letters with smooth scrolling
- **Visual Translation**: Generate beautiful shareable translation cards
- **Educational Content**: Learn practical usage with real Hebrew words
- **Responsive Design**: Works perfectly on desktop and mobile

## Tech Stack

- React 18 + TypeScript
- Vite for fast development
- Tailwind CSS for styling
- Lucide React for icons
- HTML-to-Image for sharing

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
